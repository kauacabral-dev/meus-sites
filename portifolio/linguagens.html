Linguagens que eu sei

HTML

HTML é a linguagem de marcação que utilizo para estruturar páginas web. Com HTML, consigo criar títulos, parágrafos, listas, imagens e links de forma organizada.

CSS

CSS é a linguagem que uso para estilizar minhas páginas. Com ela, posso definir cores, fontes, layouts e efeitos visuais, deixando meu portfólio mais bonito e moderno.

Python

Python é uma linguagem que utilizo para programação geral, scripts e pequenos projetos. Ela é simples e poderosa, permitindo criar desde programas básicos até automações mais complexas.

JavaScript

Tenho experiência em JavaScript para tornar minhas páginas interativas, criando animações, validações e recursos dinâmicos.

Java

Atualmente estou estudando Java, focando em lógica de programação, orientação a objetos e desenvolvimento de aplicações.
